Timer (duration)

By default, the alerts fade ha sno time, they lasts until you close one, or if other one appears (unless you set the property fadeOnNext: false).

You can set an alert time in ms by setting time: 1000 property. At this example the alert will remain 1s.

When you set a time the alert will not be closable
